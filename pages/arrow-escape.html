Visão Geral

Arrow Escape foi meu primeiríssimo contato prático com a Unity. Originalmente, desenvolvi a base desse projeto para o processo seletivo do Fellowship of the Game (FoG). Decidi revisitá-lo recentemente para aplicar o amadurecimento técnico que adquiri desde então, transformando o protótipo inicial em um jogo finalizado e estruturado.

Neste Platformer 2D de precisão, o jogador controla um carismático marshmallow que precisa calcular saltos e desviar de armadilhas para concluir as fases. O foco dessa segunda versão foi refatorar a base do código, finalizar o level design e aplicar uma nova identidade visual ao projeto.

Principais Aspectos Técnicos

Arquitetura State Machine

Primeira implementação prática de uma Máquina de Estados para controlar o jogador. Substituição de lógicas condicionais complexas por um sistema modular, separando perfeitamente os estados de pulo, corrida e queda.

Level Design com Tilesets

Construção e polimento dos cenários utilizando ferramentas de Tileset da Unity. Integração de assets gratuitos da comunidade para elevar a qualidade visual e a ambientação das fases.

Refatoração de Código Legado

Retorno a um código construído no início da minha jornada para aplicar novos conhecimentos de Engenharia de Software, transformando scripts acoplados em uma arquitetura limpa e escalável.

Mecânicas de Precisão 2D

Ajuste fino da física do Rigidbody2D e detecção de colisões para criar controles responsivos, essenciais para desviar de armadilhas e calcular saltos precisos.
